# Frontend Screenshots & Visual Guide

## Main Interface

### Home Screen
The main interface shows three review modes in the sidebar:
- GitHub PR URL
- Manual Diff
- View History

The top displays the API connection status (green checkmark when connected).

### Sidebar Configuration
The sidebar contains:
- **Review Mode Selection**: Radio buttons to choose input method
- **Severity Threshold**: Dropdown to set minimum severity
- **Analysis Categories**: Multi-select for analysis types
- **Custom Rules**: Text area for additional requirements

## Responsive Design

The interface adapts to different screen sizes:
- **Desktop**: Full width with sidebar
- **Tablet**: Stacked layout with collapsible sidebar
- **Mobile**: Single column with hamburger menu

## Accessibility Features

- High contrast colors for readability
- Keyboard navigation support
- Screen reader compatible
- Clear visual hierarchy
- Descriptive labels and help text

---

These visual representations show the clean, intuitive interface that makes code review accessible to all team members.
